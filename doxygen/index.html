Multi-Projector Project
Main Page

In this project, we split a left image and a right image of the same size from a target image. We aim to use two projectors placed in parallel to project them separately but displayed as the non-split target image by performing image processing. If we do not perform pre-image-processing, the overlap region will have color distortion and optical crack (Figure 1). To design a program capable of edge-blending, we used UML to model it, Python and third-party libraries (e.g., OpenCV, NumPy) to code, Doxygen to comment and document the code, and Redmine to manage and monitor the project. We succeeded in achieving a fast edge-blending program at the end (Figure 2).
